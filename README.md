Project Title
Intelligent Candidate Discovery & Ranking Engine

Problem Statement
Build an explainable candidate ranking engine that identifies the most suitable candidates for a given job description.

Approach
1. Parse candidate profiles
2. Extract career evidence
3. Evaluate experience fit
4. Analyze recruiter signals
5. Assess technical skill scores
6. Generate explainable rankings

Scoring Formula
Career Relevance      55%
Experience Fit        20%
Redrob Signals        10%
Skill Assessment       5%
Title Relevance        8%
Certifications         2%

Output
Top 100 ranked candidates with explainable reasoning.
